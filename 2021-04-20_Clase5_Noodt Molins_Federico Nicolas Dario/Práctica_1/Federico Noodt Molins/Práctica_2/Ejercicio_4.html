Bueno, vamos uno por uno. El ID se usa mucho para operar con lenguajes CSS, y sirve para identificar, y llamar luego al objeto.

El titulo sirve para dar una breve descripción al objeto.

El text-indent sirve para indentar el texto del párrafo.

El text-transform sirve para alterar el texto, como por ejemplo, pasar todo a minúsculas como está ahi.

Y por último, el margin es un código universal, para crear márgenes en los objetos.
